# picture-project
## 此项目实现功能是图片浏览，视频浏览，并且下载图片，下载视频，用uni-app和vue来做项目
## Project setup
```
yarn install
```

### Compiles and hot-reloads for development

```
yarn serve
```

### Compiles and minifies for production
